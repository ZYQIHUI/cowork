# Data Analysis Plan & Code Generation

Generate comprehensive data analysis plans with executable Python code for mathematical modeling tasks.

Describe the dataset, problem requirements, or modeling goals. The skill generates an analysis plan with Python code.

- Data visualization with matplotlib
- Modeling workflow (regression, classification, optimization)
- Runnable Python code with explanations
- Uses pandas, numpy, matplotlib, scipy
