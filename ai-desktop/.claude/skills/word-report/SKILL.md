# Word Lab Report Generation

Generate complete experiment/lab reports in Markdown format suitable for conversion to DOCX.

## Usage

Describe the experiment or provide data. The skill generates a structured Markdown report.

## Features

- Cover page with title, author, date
- Experiment objectives and background
- Experiment procedures and steps
- Results with data tables
- Conclusions and analysis
- Figure and table numbering with references
