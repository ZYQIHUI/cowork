# LaTeX Paper Generation

Generate complete LaTeX academic papers with proper structure and formatting.

## Usage

Provide a topic, outline, or research description. The skill generates a fully compilable LaTeX source file.

## Features

- Standard academic structure (Abstract, Introduction, Methods, Experiments, Results, Conclusion, References)
- Mathematical formula support
- Figure and table references
- Chinese language support via ctexart document class
- Directly compilable with pdflatex
