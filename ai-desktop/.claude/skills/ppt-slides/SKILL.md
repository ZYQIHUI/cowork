# PPT Presentation Generation

Generate structured presentation outlines and content.

## Usage

Provide a topic or research summary. The skill generates slide-by-slide content in Markdown format.

## Features

- Title slide with presentation name and author
- Table of contents / agenda slide
- Content slides with bullet points
- Chart and data visualization recommendations
- Summary and Q&A slides
- Slides separated by `---` for easy conversion
